## #brobots eco
![GitHub release (latest by date)](https://img.shields.io/github/v/release/brobots-hub/ecobrobotsbackend?style=flat&logo=github&labelColor=181717&color=F8F8F5)
![GitHub Workflow Status](https://img.shields.io/github/workflow/status/brobots-hub/ecobrobotsbackend/unit-tests?style=flat&logo=github&labelColor=181717&color=brightgreen)
[![#brobots eco website](https://img.shields.io/badge/%23brobots-eco-fff000)](http://eco.brobots.org.ua)

**#brobots eco** is a net of sensors that measure air quality in Brovary, Ukraine. Backend part receives live data from all active sensors and updates AQI values on the website every 15 minutes.

You can see live data at [eco.brobots.org.ua](http://eco.brobots.org.ua/).

Repositories:
- Frontend repo: [link](https://github.com/brobots-hub/ecobrobotsfrontend)
- Backend repo: [link](https://github.com/brobots-hub/ecobrobotsbackend)

## Bank of #brobots
![GitHub release (latest by date)](https://img.shields.io/github/v/release/andrew4ever/bank-of-brobots-api) ![GitHub Workflow Status](https://img.shields.io/github/workflow/status/andrew4ever/bank-of-brobots-api/Bank%20API%20CI)

[#brobots school](https://github.com/brobots-hub) has an internal currency called **brocoins**. Students get them as a reward for projects and good grades. They can buy some new hardware for next projects using them. I created a bank that allows students to manage their **brocoins** using Telegram bot.

Repositories:
- Telegram bot: [link](https://github.com/andrewyazura/bank-of-brobots-bot)
- Backend: [link](https://github.com/andrewyazura/bank-of-brobots-api)
